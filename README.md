# Zombie Trail

This project is a remix of the original Oregon Trail game, which is a text based narrative-driven game. The project was conceived by one of my students and I had led in implementing his desired features, as well as reviewing and debugging his code.

Project Highlights
– JavaScript based project built using Vite (Vue)
– uses JavaScript DOM to integrate HMTL elements into js code
– concepts used: exporting functions, event handling, DOM integration

## Instructions

1. Begin by creating a Vite project through the terminal to your local machine

2. Once the zip folder is unzipped, use the terminal to run the index.html file using 'npm run dev'

3. If you run into errors with the 'npm run dev' command, reference the links in the 'References' section below
